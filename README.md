# hseb3-db
2014\. Study project. Java desktop app with the simulation of real estate search app.
## Features
1. Object search with parameters (city, square, price)
2. Object card view
3. Managing bookmarks
## Technology
Java (AWT, Swing, Miglayout), Hibernate, PostgreSQL
